SETUP

* scan for existing APs

* if exists:
  - connect to it
* else:
  - set up as AP
  - keep track of number of connected devices



TASKS

* continuously scan for other APs, if found:
  - determine which has more connected devices
  - redirect all devices connected to other APs to that one
  - convert those APs to STA

* if STA:
  - send messages to AP
* else if AP:
  - receive messages from STAs
  - combine with own message and forward to website
